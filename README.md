KalavidaraBalaga is an Android application developed using Kotlin in Android Studio.
The app helps users discover talented artists, view their profiles, and book them for events and programs easily.

📱 Features
Artist Registration & Login
Browse Artist Profiles
Artist Booking System
Booking Management
User Feedback System
Gallery Display
Splash Screen & Authentication
Easy Navigation UI

🛠️ Built With
Kotlin
Android Studio
Firebase (if used)
RecyclerView
XML Layouts

📂 Project Structure
Activities – Handles app screens and navigation
Adapters – RecyclerView adapters
Models – Data classes
Layouts – XML UI designs
Drawable – App resources and icons

🚀 Getting Started
Prerequisites
Android Studio installed
Android SDK configured
Kotlin support enabled
Installation
Clone the repository:
git clone https://github.com/your-username/KalavidaraBalaga.git
Open the project in Android Studio
Sync Gradle files
Run the application on Emulator or Physical Device
